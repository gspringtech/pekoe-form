What makes the BAG so special is the way that a hierarchical list is rendered into a horizontal tree, with calculated spacing and padding. Unlike a normal tree, this one separates the Containers from the content. So while each level of the tree can have content, the content is only displayed for the currently selected part of the tree. I'm not sure how much of this could be done in the server. It would be a fascinating XSLT or XQuery.
